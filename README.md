# Aksaralaya — Perpustakaan Digital Pesantren

Sistem digitalisasi manajemen perpustakaan pesantren. Catat kehadiran, kelola data, dan tingkatkan semangat menuntut ilmu.

## Project info

**GitHub Repository**: https://github.com/aksaralaya/aksara-library.git

## How to Run locally

The only requirement is having Node.js & npm installed.

### Setup Steps:

```sh
# Step 1: Clone the repository
git clone https://github.com/aksaralaya/aksara-library.git

# Step 2: Navigate to the project directory
cd aksara-library-main

# Step 3: Install the necessary dependencies
# Note: If Node.js is not in your system Path (e.g., C:\Program Files\nodejs), 
# use the absolute path or update your $env:Path temporarily.
npm i

# Step 4: Start the development server
npm run dev
```

## Technologies Used
